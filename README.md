## Core API x Sistem Informasi Medis Saat Bencana

Installation Procedure

- Clone the repository
- Install & Run composer update inside the folder (composer install).
- Install composer require vlucas/phpdotenv
- Edit .env file to configure the database. (Note : For database names, match the database simsb that already exist)
